CSS

El CSS podría definirse como un tipo de lenguaje que permite definir y crear la presentación visual de un documento ya estructurado y escrito en un lenguaje de marcado como puede ser HTML.

CSS es necesario en el diseño web y la programación frontend para crear experiencias web dinámicas y atractivas. Por ejemplo, CSS ofrece herramientas para aplicar transformaciones y animaciones a elementos.

JavaScript

JavaScript es un lenguaje de secuencias de comandos que te permite crear contenido de actualización dinámica, controlar multimedia, animar imágenes y prácticamente todo lo demás.
